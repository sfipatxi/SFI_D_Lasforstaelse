SFI KURS D - SJÄLVRÄTTANDE UPPGIFTSPAKET "NATUR OCH MILJÖ"

INNEHÅL:
========
- index.html                     Startsida med navigation till alla uppgifter
- uppgift_1_artikel_frågor.html  Läsförståelse - multipel choice (6 frågor)
- uppgift_2_understrukna_ord.html Ordförståelse (6 ord i kontext)
- uppgift_3_para_ihop.html       Para ihop åtgärder med syften (5 par)
- uppgift_4_meddelanden.html     Läsa meddelanden och svara på frågor (8 frågor)
- uppgift_5_delade_meningar.html Halvsanna påståenden (6 frågor)
- uppgift_6_åsikter.html         Läsarreaktioner på insändare (6 frågor)
- total_results.html             Slutlig resultatsammanfattning
- artikel_natur_miljö.txt        Källtexten för uppgifterna
- README.txt                     Denna fil

FUNKTIONALITET:
===============
✓ Sparning av svar i sessionStorage (försvinner när webbläsarfönstret stängs)
✓ Feedback visas direkt efter submission (rätt/fel)
✓ Resultatet visas BARA om alla frågor är besvarade
✓ Inputs disabler efter submission (kan inte ändra svar)
✓ Feedback visas vid återbesök (ser tidigare svar + om de var rätt/fel)
✓ Scroll till resultat automatiskt
✓ Omedelbar validering med felmeddelanden för tomma svar
✓ Navigationsknappar mellan uppgifter
✓ Totalt resultat för hela provet på sista sidan

ANVÄNDNING:
===========
1. Öppna index.html i en webbläsare
2. Klicka på en uppgift för att börja
3. Svara på alla frågor
4. Klicka "Kontrollera dina svar"
5. Se resultatet
6. Navigera till nästa uppgift eller tillbaka till start
7. Efter uppgift 6 - klicka "Visa totalt resultat" för sammanfattning

TEKNISK INFO:
==============
- HTML5 + Vanilla JavaScript (ingen externa beroenden)
- sessionStorage för temporär datalagrind
- Responsive design (fungerar på mobil, surfplatta, dator)
- Färgschema: grön #27ae60 (rätt), röd #c0392b (fel), blå #3498db (knappar)
- Kompatibel med alla moderna webbläsare

NIVÅ:
=====
SFI Kurs D - Läsförståelse på avancerad nivå

TIPS FÖR LÄRARE:
================
- Provet kan göras i en session eller sprida över flera tillfällen
- Svaren sparas automatiskt mellan uppgifter
- Efter att ha gjort hele provet kan eleverna starta på nytt (sessionStorage rensas när webbläsarfönstret stängs)
- Utskrift av resultatet kan göras via webbläsarens print-funktion
